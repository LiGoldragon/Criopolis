## Where the four converge

Strong consensus on three points:

### 1. Devil should transform, not stay or be deleted

All four seats agree the courtroom-devil-finding-flaws posture is
wrong for positive research. Names proposed:

| Seat       | Proposed name                | Function                                  |
| ---------- | ---------------------------- | ----------------------------------------- |
| Aesthete   | `stranger` (Plato's xenos)   | aporia-bringer; reads as if first-time    |
| Theorist   | (no rename — function only)  | "resist closure"                          |
| Pragmatist | `stranger / examiner`        | first-contact reading; pressure on consensus |
| Devil      | `gadfly` (Apology 30e)       | counter-horizon; premature-closure prevention |

The function-set across the four is the same: *resist premature
wholeness*. The differences are nominal.

### 2. Asymmetric prompts

All four say each seat's prompt should have a different *shape*,
not merely different content. Pragmatist sharpens: **symmetric
output contract** (predictable shape so synthesis is cheap),
**asymmetric inner method**.

Each seat opens differently:

| Seat       | Opening move        | Why                                  |
| ---------- | ------------------- | ------------------------------------ |
| Aesthete   | quote               | beauty is encountered                |
| Theorist   | definition          | truth is stated                      |
| Pragmatist | scenario            | the good is enacted                  |
| Gadfly     | question            | aporia is asked                      |
| Mayor      | verdict-shape       | synthesis is spoken                  |

A reader skimming a reply should predict the seat from structural
shape alone.

### 3. Pace = Aristotle's μέσον

All four cite or echo *Nicomachean Ethics* II.6, II.9.1109a24 (the
mean). Pragmatist and aesthete give compatible numbers:

- **Ordinary research bead:** 60–120 minutes (pragmatist), 2–4
  hours for shape-setting (aesthete + pragmatist agree).
- **Shortest worthwhile reply:** ~350–500 words (pragmatist).
- **Longest useful ordinary reply:** ~2000 words (pragmatist).
- **Cadence:** one bead per question, however long the question
  takes (everyone). The unit is the *question*, not the
  calendar.
- **Tell-of-enough:** removing the strongest citation would
  *collapse* the position, not merely *trim* it (aesthete).
  The reply names what would falsify it (theorist).

## Where they disagree — cardinality

Real split: 2 say four seats, 2 say five.

- **4 seats + mayor** (theorist, pragmatist): minimum sufficient
  move-set — *state, ground, test, resist-closure*. 5+ adds
  redundancy or requires a meta-role. Source-rigor handled by
  the librarian-as-service, not as a forum seat.
- **5 seats + mayor** (aesthete: with `stranger` as 5th in a
  Platonic-transcendentals mandala; devil: with both `gadfly` and
  `archivist` separating premature-closure-resistance from
  citation-discipline).

## My stance — 4 seats, devil → gadfly

I take theorist + pragmatist's view. **Four seats plus mayor as
orchestrator (not a voice).** Devil renamed to **gadfly**.

Reasoning:

1. **The four-move set covers it.** State (aesthete), ground
   (theorist), test (pragmatist), resist closure (gadfly). Every
   substantive failure mode I've watched in this session — false
   unanimity, premature wholeness assertion, source drift — falls
   under one of these four functions. The fifth move (archivist /
   stranger) is real but is a *technical* function the librarian
   already discharges, not a *philosophical* one needing a seat.
2. **Gadfly is the most defended single name.** Devil chose it
   themselves with Plato (*Apology* 30e), Patanjali (II.33,
   *pratipaksha-bhavana*), Ptolemy (I.2 conjectural humility),
   and Spivak (olog granularity / "evaporate into nothingness")
   as grounds. Stranger is also strong but less function-clear.
3. **Aesthete's mandala-of-five is beautiful but not load-
   bearing.** The Platonic-transcendentals frame
   (Beautiful/True/Good/Question/One) is a recovered structure,
   not an invented one. But the *fifth* — Question — is what
   gadfly already does. Naming it stranger and seating it
   separately doesn't add discriminatory power; it adds reading
   cost.
4. **Devil's archivist concern is real but mis-located.** The
   forum can drift on citations — the prior wholeness round had
   exactly this problem. The fix is for the librarian to be a
   *service* the seats use (book-add via mail), not a fifth seat
   on every bead.

If wholeness research surfaces real source-drift the librarian
can't catch, revisit. For now: 4 + mayor.

## Proposed structure

```
Forum:
  aesthete    — STATE; opens with quote; beauty / form / friction-free reading
  theorist    — GROUND; opens with definition; derivation / axioms
  pragmatist  — TEST; opens with scenario; cost / blast / decision
  gadfly      — RESIST CLOSURE; opens with question; horizon / counter-example

Orchestrator:
  mayor       — SYNTHESIZE; opens with verdict-shape; routing + final responsibility

Service:
  librarian   — fetches + catalogs; not a forum seat
```
